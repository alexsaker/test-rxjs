# test-rxjs

This is a small test project to help me test rxjs functions with webpack.

## Getting Started

In order to use it, you can clone the project
```
git clone https://github.com/alexsaker/test-rxjs
```
and install it
```
npm install
```

### Prerequisites

No prerequisites needed

### Running

To Build
```
npm run build:dev
```

To Serve
```
npm run serve:dev
```

To Build & Serve
```
npm run watch:dev
```
